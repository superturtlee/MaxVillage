# MaxVillage（最大村庄）

一个 Minecraft 1.16.5 Forge 模组，允许你通过配置文件调整自然生成建筑的生成频率和最小距离。

## 功能特性

- **可配置的建筑间距**：调整建筑生成的频率（以区块为单位）
- **可配置的建筑分离度**：设置建筑之间的最小距离（以区块为单位）
- **支持多种建筑类型**：
  - 村庄
  - 沙漠神殿
  - 丛林神庙
  - 沼泽小屋（女巫小屋）
  - 雪屋
  - 海底神殿
  - 沉船
  - 海底废墟
  - 掠夺者前哨站
  - 林地府邸
  - 下界要塞
  - 末地城
- **易于使用的 TOML 配置文件**
- **支持服务端和客户端**

## 安装方法

1. 下载并安装 Minecraft Forge 1.16.5（版本 36.2.42 或兼容版本）
2. 下载 MaxVillage 模组 JAR 文件
3. 将 JAR 文件放入你的 Minecraft `mods` 文件夹
4. 运行 Minecraft 以生成默认配置文件
5. 编辑配置文件 `config/maxvillage-common.toml`
6. 重启 Minecraft 使更改生效

## 配置说明

首次运行后，你会在以下位置找到配置文件：
```
<minecraft目录>/config/maxvillage-common.toml
```

详细的配置说明请参见 [CONFIG_GUIDE.md](CONFIG_GUIDE.md)（英文）

### 快速示例

让村庄生成更频繁：
```toml
[structures.village]
    spacing = 16     # 村庄间距更近（默认：32）
    separation = 4   # 可以更靠近生成（默认：8）
```

## 配置参数说明

每种建筑类型都有两个关键参数：

- **spacing（间距）**：建筑之间的平均距离（以区块为单位）。数值越大，建筑生成越稀少。
- **separation（分离度）**：建筑之间的最小距离（以区块为单位）。确保建筑不会生成得太近。

## 从源代码构建

要求：
- Java 8 JDK

步骤：
```bash
git clone https://github.com/superturtlee/MaxVillage.git
cd MaxVillage
./gradlew build
```

构建的 JAR 文件将位于 `build/libs/` 目录

## 使用提示

### 让村庄更常见
```toml
[structures.village]
    spacing = 16    # 将默认间距减半
    separation = 4  # 将默认分离度减半
```

### 让村庄更稀有
```toml
[structures.village]
    spacing = 64    # 将默认间距加倍
    separation = 16 # 将默认分离度加倍
```

### 让林地府邸更容易找到
林地府邸默认非常稀有。让它们更容易找到：
```toml
[structures.mansion]
    spacing = 40    # 减半默认间距
    separation = 10 # 减半默认分离度
```

## 重要注意事项

- **区块大小**：Minecraft 中的一个区块是 16x16 方块
- **世界生成**：这些设置只影响尚未生成的新区块
- **现有世界**：要在现有世界中看到变化，需要探索尚未生成的新区域
- **多人游戏**：使用服务器的配置文件；客户端配置将被忽略
- **有效范围**：所有间距和分离度值必须在 1 到 1000 之间

## 常见问题

**问：我的更改没有生效！**
- 确保编辑配置后重启了 Minecraft
- 记住现有区块不会改变 - 需要探索新区域
- 检查你的值是否在有效范围内（1-1000）

**问：能完全禁用某个建筑吗？**
- 将分离度设置为等于或大于间距会让建筑变得极其稀有，但不会完全禁用

**问：这在多人服务器上能用吗？**
- 可以！服务器管理员需要安装并配置此模组。所有连接的客户端将使用服务器的设置。
